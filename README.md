# generator
* Запуск без аргументов – программа переходит в интерактивный режим, последовательно запрашивая количество записей, формат и имя файла.
 Пример:
 python generator_2.py
* === Генератор тестовых данных ===
- Количество записей (по умолчанию 10): 5
- Формат сохранения (csv/json, по умолчанию csv): json
- Имя файла без расширения (по умолчанию data): my_data

* Запуск с аргументами – использует переданные параметры. Если какой-то аргумент опущен, подставляется значение по умолчанию.
 Пример:
 python generator_2.py -n 20 -f csv -o output

* Генерация email – строится на основе имени, фамилии, города, телефона и даты рождения (см. функцию generate_email). Email всегда валиден.
* Валидация – все записи проверяются на корректность, результаты выводятся в консоль.
* Сохранение – данные записываются в CSV или JSON с указанным именем файла.
* Установка зависимостей:
- pip install faker
- pip install transliterate
